## Local에서 demo 실행하는 법
### env 설정
`.streamlit/secrets.toml` 생성 후 다음의 env 작성 
```
OPENAI_API_KEY = ""
STARDOG_ENDPOINT = ""
STARDOG_USERNAME = ""
STARDOG_PASSWORD = ""
STARDOG_DATABASE = ""
```

### 가상환경
```
conda create -n devship-demo python=3.11
conda activate devship-demo
pip install -r requirements.txt
```
### Streamlit 실행
```
streamlit run demo.py
```
[http://localhost:8501/](http://localhost:8501/) 로 접속
